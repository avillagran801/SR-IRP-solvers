Detailed Description

Pairs a raw generated sequence with its fully decoded solution.

This struct is used by daily solvers to return both the abstract algorithmic representation (like a GA chromosome or an ACO ant path) and the actual evaluated business logic (the decoded routes, times, and inventory).
