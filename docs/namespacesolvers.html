Executes the Version 1 Ant Colony Optimization (ACO) solver.

This specific solver pipeline combines:

  • Multi-Period Strategy: Standard Beam Search (evaluates nodes based on isolated daily utility).
  • Daily Algorithm: Standard Ant Colony Optimization (ACO).
  • Decoder: Standard Greedy Decoder (demand fulfillment followed by capacity maximization).
Parameters
instance_nameThe relative or absolute file path to the problem instance text file.
max_time_secondsTotal multi-period time budget in seconds (default: 90).
beam_widthNumber of candidate states to preserve per day in the Beam Search (default: 2).
num_antsNumber of ants used per iteration in the daily ACO (default: 200).
alphaPheromone trail weight; dictates reliance on past experience (default: 1.0).
betaHeuristic information weight; dictates reliance on greedy choices (default: 1.0).
qPheromone deposit constant (default: 90.0).
evaporation_rateRate at which pheromones evaporate per iteration, from 0.0 to 1.0 (default: 0.89).
Returns
common::SolverResult An object containing the decoded final routes, multi-period inventory states, utility metrics, and execution time.
