Detailed Description

Ant Colony Optimization solver for the daily routing and inventory problem that considers previous profit.

This class generates daily routes by simulating the behavior of ants finding the shortest paths, guided by pheromone trails and heuristic evaluations, decoded via the provided DecoderInterface.
