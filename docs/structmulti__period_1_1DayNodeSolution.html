Detailed Description

Represents a single candidate state (node) in the multi-period search tree.

During the Beam Search process, multiple candidate solutions are generated for each day. This struct encapsulates a specific candidate, tracking its lineage to reconstruct the full multi-period path, the raw sequence generated by the daily meta-heuristic (GA/ACO), and the fully evaluated day's solution.
