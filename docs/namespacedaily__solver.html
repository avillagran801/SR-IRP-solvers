daily_solver::SolverAlgorithm

A generic function signature for any daily routing solver.

This type alias acts as a universal contract. It allows higher-level multi-period strategies (like Beam Search) to accept any daily solver (such as GA or ACO) as a lambda or callback parameter, completely decoupling the multi-period logic from the specific daily optimization technique.

Parameters
ctxThe context containing the instance data and current day's starting state.
time_limit_secondsThe maximum execution time allowed for the daily solver.
Returns
A vector of candidate DailyResult solutions.
